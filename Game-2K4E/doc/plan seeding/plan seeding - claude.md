## Quy trình Seeding

### Seeding ở đâu?

**Ưu tiên cao:**

- Facebook Groups: nhóm game mobile Việt Nam, nhóm meme/trend, nhóm fandom của các celeb/show đang được dùng làm album ảnh trong game
- TikTok comment section: các video trending liên quan đến chủ đề album đang chạy trong game
- Reddit r/Vietnam, r/vnhardware (nếu target rộng hơn)

**Ưu tiên trung bình:**

- Threads, X (Twitter VN)
- Discord server gaming Việt

---

### Seeding bằng cái gì?

| Dạng nội dung           | Ví dụ cụ thể                                                                |
| ----------------------- | --------------------------------------------------------------------------- |
| Kể chuyện tự nhiên      | "Hôm nay tình cờ chơi cái game gộp ảnh [tên], lag não vãi 😭 ai chơi chưa?" |
| Flex điểm số            | Chụp màn hình bảng xếp hạng + caption thách đấu                             |
| Reaction về album       | "Ủa sao nó lại dùng ảnh [celeb X] cho game này 💀 hợp lý vãi"               |
| Câu hỏi giả vờ hỏi thật | "Ai biết game gộp ảnh kiểu 2048 không, mình đang tìm cái hay hay chơi"      |

---

### Quy trình cụ thể

**Giai đoạn 1 — Pre-launch:**

1. Chuẩn bị 3–5 tài khoản seeding có lịch sử đăng bài tự nhiên (không dùng nick mới toanh)
2. Tuần 1: Đăng dạng "hỏi thật" ở các group game → tài khoản khác vào trả lời giới thiệu game
3. Tuần 2: Khi bài tò mò "Gộp A + B = ?" được đăng trên fanpage → seeding kéo traffic bằng cách tag bạn bè vào comment tự nhiên
4. Tuần 2–3: Comment vào các bài trend liên quan đến chủ đề album đang teaser

**Giai đoạn 2 — Launch:**

1. Seeding trải nghiệm thật: chụp màn hình gameplay, điểm số, khoảnh khắc "gộp được ô cao" → đăng lên story/group
2. Phản ứng về event top bảng xếp hạng: "Giải thưởng ngon thế này mà không chơi là phí 👀"
3. Nếu có người thật đang đăng về game → comment/share ngay để khuếch đại

---

### Nguyên tắc seeding hiệu quả

- **Không lộ liễu**: Tránh kiểu "Game hay lắm mn ơi link tải đây nè" → bị nhận ra ngay
- **Bám theo album đang hot**: Nếu album đang dùng ảnh liên quan đến drama/show đang viral → seeding đúng vào cộng đồng fandom đó
- **Tỉ lệ 70/30**: 70% nội dung seeding là cảm xúc/câu chuyện, 30% mới là thông tin game
